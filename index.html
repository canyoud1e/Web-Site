Семантична структура сучасного HTML

Опубліковано 11 лютого 2026

Роль глобальних орієнтирів

Глобальні орієнтири визначають базову архітектуру документа. До них належать елементи <header>, <nav>, <main>, <aside> та <footer>.

«Семантичні елементи надають значення структурі документа, роблячи його зрозумілішим для людей і машин».

Вкладеність та ієрархія

Кожен <article> може містити власні <section>, що формують логічні підрозділи. Важливо дотримуватися послідовності заголовків: від h1 до h3 без пропусків рівнів.

Контентні одиниці як автономні модулі

Опубліковано 28 січня 2026

Що таке автономний допис?

Автономний допис — це самостійний інформаційний блок, який може існувати поза межами сторінки (наприклад, у RSS-стрічці). Саме тому для нього використовується тег <article>.

Допоміжні елементи

Для уточнення інформації використовуються спеціальні теги: <time> для дат, <address> для контактів, <blockquote> для цитат.

Правильна семантика — це інвестиція в доступність та SEO.
